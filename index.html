Projects

A GAN (Generative Adversarial Network)-Based Framework for Multi-Class Skin Disease Classification

Developed a deep learning model using Generative Adversarial Networks (GANs) for multi-class skin disease classification on the HAM10000 dataset. The generator creates synthetic skin lesion images to augment and balance the dataset, while the discriminator simultaneously distinguishes between real and synthetic images and classifies them into disease categories. This approach addressed limitations of limited and imbalanced medical data, resulting in improved model performance and diagnostic accuracy. The project demonstrates the potential of GANs in medical imaging by supporting more reliable AI-assisted dermatological diagnostics.

Microcontroller Based Alcohol Detection Alert & Car Engine Blocking System

Designed a microcontroller-based safety system using ATmega328P and MQ-3 gas sensor to detect alcohol from the driver’s breath. The system alerts passengers via LCD, buzzer, and LED, and automatically blocks the car engine if alcohol is detected. The engine resumes only when the driver is sober.

Smart Alcohol Detection and Alert System (IoT-Based)

Designed and implemented an IoT-enabled alcohol detection system using an ESP8266 microcontroller, MQ3 gas sensor, and GPS module. The system detects elevated alcohol levels and responds by halting a motor and sending real-time alerts via ThingSpeak and IFTTT. It also logs sensor readings along with location data for remote monitoring and safety analysis, aiming to reduce risks associated with drunk driving.
